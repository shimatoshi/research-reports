# Research Reports

調査・研究レポートのアーカイブ。

## フォルダ構成

| フォルダ | 内容 |
|----------|------|
| `biology/` | 生物学 |
| `history/` | 歴史学 |
| `engineering/` | エンジニアリング |
| `engineering/knowledge/` | 実証済みの技術知見 |
| `toolbox/` | 再利用可能なツール群 |

## ソース分類（biology共通）

各トピックの下に以下のフォルダがある:

| フォルダ | 基準 |
|----------|------|
| `papers/` | 学術論文がソース |
| `field/` | 自分がフィールドに出て得た体験・観察 |
| `web-verified/` | ネット等の情報だが学術論文で裏付けが取れたもの |
| `unverified/` | ソースはあるが根拠が不明・未検証の情報 |
