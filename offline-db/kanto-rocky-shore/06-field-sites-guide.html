関東 磯・岩礁海岸 採集/観察サイトガイド

在野研究者向け現地ポケット資料 (2026-04 編纂)

本書は相模湾〜東京湾〜外房という関東の主要磯相をカバーする現地行動マニュアルである。前章 (01-geography.md〜05-reference-links.md) で整理した生物相・微生息地区分の知見を実際に現地で展開するために、(a) 潮汐の読み方、(b) 各サイトのアクセス/規模/危険/漁業権情報、(c) 観察記録テンプレを一つにまとめた。個人研究目的の採集・観察を前提としており、商業採捕・密漁は本書の範囲外である。

0. 潮汐とタイミング — まずこれを押さえる

磯観察の成否は「大潮の干潮に現地にいるか否か」で9割決まる。採集機会は1サイトあたり年間で大潮干潮が昼間に来る日＝実質30-40日程度しかない。

0.1 気象庁潮位表の使い方

  • 入口 → 気象庁 潮位表トップ。関東は 関東・伊豆諸島ページ から地点選択。
  • 基準観測点の目安:
  • 相模湾〜三浦西岸: 油壺 (AB) または 横須賀 (YK)
  • 東京湾内 (走水/観音崎): 横浜 (QS)
  • 外房 (勝浦・鴨川): 勝浦 (KZ)
  • 内房 (富津・鋸南): 千葉 (CH) または 木更津
  • 読み方: 予測は大潮の平均干潮面を 0 cm とする「天文潮位」。台風・低気圧時は実測が±30 cm ずれることがあるため、出発前日に リアルタイム観測データ と照合する。
  • 「最低潮位」記号が月に1-2回付く日が勝負日。潮位が-10 cm を切る日を狙えば普段水没している低潮帯下限が露出する。

0.2 tide736.net (民間の代替)

tide736.net は全国736港をカバーし、UI がスマホで扱いやすい。気象庁と同じ調和定数ベースだが、 - グラフ表示で視覚的に干満差が分かる - 日の出/日の入り重ね表示があり「昼大潮かどうか」が一目で判断可 - 主要磯観察地 (城ヶ島、油壺、三ツ石、鵜原、布良 (館山) 等) の単独ページがある

オフライン派は tide736 の月次ページをブラウザで保存 (single-file HTML) しておくと通信圏外でも参照可能。

0.3 ベストウィンドウ

  • 干潮時刻の前後 90 分 (合計約3時間) が観察可能時間
  • 干潮の30分前到着で、徐々に露出していく磯を順に観察するのが定石
  • 満潮に変わった瞬間が最も危険 — 潮が戻る速度は早く、沖ノ根で孤立する事故が多い

1. 装備

1.1 必須

  • マリンシューズ (フェルトピン底): サーフ系ではなく磯釣り用。ビーサン/クロックスは論外。濡れた海藻で滑落骨折の常連
  • グローブ: 軍手は不可 (ケガに弱く濡れると脱げる)。滑り止めゴム付き磯釣り用グローブ推奨
  • 採集バケツ: オモリ付きロープ付きの磯バケツ。海中で水を汲みやすく、海水循環に使える
  • 観察ケース: 透明プラケース (100均) 2個 — 被写体を入れて腹側撮影用
  • スポイト / 細ピンセット: タイドプール底の堆積物からヨコエビ類・カニダマシ幼体を拾う
  • ライフジャケット: 単独行では必ず着用。特に鵜原理想郷、城ヶ崎のような外洋露出度の高いサイト
  • 防水スマホケース + 予備バッテリー: 撮影と位置記録

1.2 あると便利

  • 水中メガネ/シュノーケル: 水深のあるタイドプール底の観察用
  • 偏光サングラス: 水面反射を切り、魚類行動観察の必需品
  • 小型 LED: 岩陰照射用。ギンポ類・ハゼ類の見つけ率が跳ね上がる
  • ノギス/定規: 計測記録用 (種同定時の前鰓蓋棘配置等に必要)
  • 屈折計 (小型): 塩分計測用。河口近接サイトでは有意
  • 1/100,000 地形図または YAMAP オフライン地図: GPS 圏外対策

1.3 服装

長袖・長ズボン必須 (岩角と有毒生物から保護)。ラッシュガード + クイックドライロングパンツが標準。夏でも磯は日陰/風で冷えるので薄手フリース予備。

3. 三浦エリア (最重要)

3.1 城ヶ島

  • アクセス: 京急三崎口駅 → 京急バス「城ヶ島」行き35分 → 終点下車。または東側「白秋碑前」。西側 (白秋碑前寄り) 公共 P が磯への距離最短。
  • 磯の範囲: 島全周だが観察実戦区は長津呂の磯 (灯台奥、ひな壇状岩棚) と馬の背洞門〜赤羽根崎。東西併せて約2km、大潮時の有効露出は累計500m超。
  • 大潮時露出: 潮位-20 cm で長津呂外縁の赤褐色藻類帯まで露出、-5 cm でも十分楽しめる。
  • 代表観察対象: ウミウシ類 (アオ・シロ・ヒロ系多数)、クモヒトデ類、イソスジエビ、ギンポ類、メジナ幼魚、巨大ヒザラガイ。
  • 危険: 馬の背洞門の北側は波あたり強 — 大潮満潮時の戻りで孤立事故多発。島南岸は風が上がると即中止。
  • 採集可否: 三崎漁協の共同漁業権エリア。イセエビ/サザエ/アワビ/タコ/ナマコ/海藻は全面不可。小型甲殻類・ハゼ類の研究目的観察は可。県条例 磯遊びのルール 参照。
  • 紹介: miurahantou.jp 城ヶ島 / shizengate 城ヶ島 / 個人ブログ Innocent Clovers
  • 適期: 4-6月 (海藻着生ピーク)、9-10月 (稚魚加入)。真夏は観察者過多。

3.2 荒崎海岸

  • アクセス: 三崎口駅から京急バス「荒崎」行き約20分。荒崎公園 P あり。
  • 磯の範囲: 公園南側「どんどんびき」〜「潮見穴」一帯、約400m。平坦な凝灰質泥岩の棚が続く。
  • 大潮時露出: 潮位 0 cm で棚全体、-10 cm でオーバーハング下のウニ溝まで到達可。
  • 代表観察対象: ウメボシイソギンチャク、ニホンクモヒトデ、ケヤリムシ類、多種ハゼ、磯の貝類多彩。
  • 危険: 台風残波でうねりが棚越しに上がる。風速5 m以上で中止推奨。
  • 採集可否: 三浦市漁協管轄。共同漁業権対象全種禁止。
  • 紹介: shizengate 荒崎 / asobii.net 荒崎 / tokyo-snorkeling.com 荒崎
  • 適期: 5-7月、10月。

3.3 立石公園・秋谷

  • アクセス: JR逗子駅 → 京急バス「立石」下車すぐ。無料 P (62台) 24時間。
  • 磯の範囲: 立石 (高さ12 m巨岩) 北〜南側、約150 m。狭いが駐車から30秒。
  • 大潮時露出: 潮位+10 cm でも十分観察可。エントリー最容易。
  • 代表観察対象: ウメボシイソギンチャク群落、フジツボ帯、イソガニ、各種巻貝。初心者・子連れ入門に最適。
  • 危険: 少。ただし北側の岩盤はノリが乗ると滑る。
  • 採集可否: 横須賀市漁協。横須賀市の注意 に明記の通りアワビ/サザエ等禁止。
  • 紹介: shonan-miura tateishi / miurahantou 立石
  • 適期: 通年だが夏場は人出過多。

3.4 油壺

  • アクセス: 三崎口駅 → バス15分 → 油壺バス停 → 徒歩10分。東京大学臨海実験所下の磯 (諸磯湾口) が観察実戦区。
  • 磯の範囲: 湾内静穏磯〜湾口外洋性磯まで多様。約300m。
  • 大潮時露出: -5 cm で実験所前の岩棚全域露出。
  • 代表観察対象: 500種以上の採集記録を持つ歴史的観察地。マナマコ、トコブシ、多様な棘皮動物、ミドリイシ系サンゴ北限域。
  • 危険: 少 (湾内静穏)。ただし磯釣り客と輻輳しがち。
  • 採集可否: 三崎漁協水域かつ実験所保全区域。学術利用は事前に臨海実験所に相談が望ましい。一般採集は共同漁業権対象全種禁止。
  • 紹介: 東京大学三崎臨海実験所公式 / note 門田和雄 三崎臨海 / 釣り場案内人サトシ 油壺
  • 適期: 5-10月。

3.5 剱崎

  • アクセス: 京急三浦海岸駅 → 徒歩30分、または松輪バス停。
  • 磯の範囲: 剱崎灯台下〜江奈湾手前、関東ふれあいの道「三浦・岩礁のみち」 の起点。1 km超。
  • 代表観察対象: 外洋性相。ヨメガカサ、オオクボクガイ、タイドプール底にはギンポ類・オヤビッチャ迷入個体。
  • 危険: 高。うねり直撃エリアで過去に死亡事故複数。単独行非推奨。
  • 採集可否: 松輪漁協エリア。
  • 紹介: miura-city 岩礁のみち / kojitusanso 岩礁のみち
  • 適期: 晴天無風の5-6月、10-11月。

3.6 観音崎・走水

観音崎は 県立観音崎公園 直下の磯で東京湾内の内湾相が観察できる希少サイト。観音崎自然博物館 との連携観察会あり。走水は 走水海岸 — アサリ主体で磯相は弱いが、運河性の細い岩礁がある。両地点とも潮位差が三浦外洋より小さく、干潮でも露出は控えめ。

3.7 久里浜・野比

野比海岸 は磯というより活断層露頭と新第三紀地層の観察地。生物観察優先度は低いが、地質コンテキスト学習には有用。

4. 葉山〜鎌倉

4.1 葉山 芝崎 (あぐい島周辺)

  • アクセス: JR逗子駅 → バス「海岸回り」→「海岸通り」下車。葉山しおさい博物館 併設。
  • 磯の範囲: 芝崎岬の潮間帯約200m。葉山町観光協会が 潮だまり観察コース を公式に整備。
  • 代表観察対象: 昭和天皇採集の歴史的生物相。ウミウシ類が特に濃い (note 海遊びのほや 芝崎編)。
  • 採集可否: 「天皇海岸」として知られ保全意識高い。採集は控え観察主体で。共同漁業権対象種不可。
  • 紹介: しおさい博物館、note 芝崎編
  • 適期: 4-6月 (ウミウシ繁殖期)。

4.2 森戸・稲村ヶ崎

森戸は 砂浜中心 で磯は岬周辺のみ。稲村ヶ崎は狭いが江ノ電駅徒歩3分で鎌倉材木座の和賀江嶋 (日本最古の築港跡、鎌倉海浜公園資料) とのセット行程が可能。

5. 真鶴半島 (箱根ジオパーク)

5.1 三ツ石海岸

  • アクセス: JR真鶴駅 → バス「ケープ真鶴」下車 → 遊歩道下り10分。
  • 磯の範囲: 半島先端3岩礁基部〜西側磯、約400m。西側タイドプールが濃い。
  • 大潮時露出: 春〜秋の大潮で陸繋島化し三ツ石本体まで歩いて渡れる。
  • 代表観察対象: ウメボシイソギンチャク、ナマコ類、ハゼ類、ブダイ幼魚、ウニ類多種。
  • 危険: 陸繋化後の満ち戻りで孤立事故多発。時計で厳密管理。
  • 採集可否: 真鶴町の「お林」由来の保全文化が強く、生物採集は強く自粛推奨。国際基準「生物多様性の観点から重要度の高い海域」指定。
  • 紹介: 町公式 / 遠藤貝類博物館 / 箱根ジオパーク note / 海遊びのほや #1 / Tabist Magazine
  • 適期: 4-10月、特に大潮陸繋化の日。

5.2 琴ヶ浜・番場浦

西岸の2点。琴ヶ浜 は階段/トイレ整備され初心者向け、番場浦 はブダイ釣りの聖地で大型タイドプールにサザエ大型個体 (ただし漁業権対象)。

6. 湘南〜伊豆北東

6.1 江ノ島 南磯 (湘南港タイドプール)

人工構造物だが外海と壁で仕切られ初心者・子連れに最も安全。無料・8-18時・asobii.net 解説。神奈川県立博物館報告 (上田ら 江の島タイドプール魚類) が種組成を公表しており、在野研究のベースラインとして使いやすい。

6.2 茅ヶ崎 烏帽子岩

沖磯、渡船必要。たつみ釣具店 等から渡船。磯釣り中心で一般磯観察には非実用的だが、姥島群礁の孤立岩礁生物相はユニーク。

6.3 湯河原 福浦・網代

福浦漁港 は漁港内の岩壁観察向き。網代は熱海市内の 漁港町、磯観察は主要目的にはならない。

6.4 伊東 城ヶ崎海岸

伊豆北東の主力磯。柱状節理の海食地形でタイドプールの水深が深く、魚類相が関東本土側よりも南方性を帯びる (beach-izu.com)。汐吹公園・橋立吊橋下が観察実戦区。

7. 房総

7.1 勝浦 鵜原理想郷

  • アクセス: JR鵜原駅徒歩10分 (トンネル抜け坂下る)。
  • 磯の範囲: リアス海岸の入り江間で複数の小磯が連続、約1km。
  • 代表観察対象: 外房外洋性相。毛戸浦の干潮磯が濃い (房州仙人 YAMAP 活動日記)。
  • 危険: 外洋うねり直撃。単独非推奨。
  • 採集可否: 南房総国定公園内。景勝地指定かつ勝浦漁協。共同漁業権対象禁止。
  • 紹介: 勝浦市観光 / 房総ism / clasis.jp
  • 適期: 4-6月、10-11月。

7.2 鴨川・小湊/内浦

千葉県立鴨川青少年自然の家 が教育プログラム運営。内浦海岸 は静穏で子供連れ・家族研究向き。市場魚貝類図鑑 外房磯遊び に代表種記載。

7.3 太海・守谷

守谷海岸 は「日本の海水浴場88選」「水質 AA」。JR上総興津駅徒歩7分で車なし研究者にも好適。

7.4 富津岬

内湾干潟寄り。露岩は少なく、富津岬 Wikipedia によれば南側は外洋防波堤、北側は 174 ha の干潟。磯より干潟性観察優先。

7.5 鋸南・保田

鋸南町公式 によれば勝山・保田両漁協の監視区域。貝類・海藻採取禁止、潜水器具・スピアガン禁止が明示。観察主体で利用。

8. 漁業権・法令セクション

8.1 基本ルール

水産庁 密漁対策・政府広報オンライン・神奈川県磯遊びルール の三点を必ず事前参照。

8.2 「第一種共同漁業権」対象の主要種

在野研究者の採集でも全面禁止。研究目的でも許可なく採ると100万円以下の罰金。

  • イセエビ (全国で共同漁業権対象)
  • アワビ (2020改正漁業法で「特定水産動植物」指定、違反は3年以下拘禁刑/3,000万円以下)
  • サザエ
  • タコ (定着性)
  • ナマコ (アワビと同じく特定水産動植物)
  • シラスウナギ (同上)
  • ワカメ・コンブ・テングサ等の海藻類

8.3 使用禁止漁具 (神奈川県)

  • カニ網
  • 忍者くまで (網/カゴ付)
  • スピアガン/水中銃
  • 潜水器具を使った採捕 (素潜り可、スクーバは不可)

竿釣り・手釣り・タモ網のみが遊漁者 (非漁業者) の合法漁具。

8.4 国立/国定公園規制

  • 南房総国定公園 (鵜原〜館山): 景観改変不可、許可なき動植物採取制限。
  • 真鶴半島 (町条例「お林」保護区): 採集自粛文化。
  • 箱根ジオパーク (真鶴含): 地質学的対象物持ち出し禁止。

12. 在野研究の実務アドバイス (編者注)

  1. 「大潮カレンダー」を作る: 年初に tide736 で各基準港の昼間大潮日を全て拾い、週末と合致する日を年10〜15日確保。この日は予定を入れない。
  2. 写真は腹背2面+斜位1面を標準化: 将来の同定で腹側情報が欠けて後悔するパターン回避。
  3. 生物は採集せず撮影優先: 漁業権グレー地帯でのトラブル回避、および個体群撹乱回避。真に必要な分類標本のみ、各県水産事務所に遊漁者から研究者登録の相談をしておくと説得力が上がる。
  4. 同一サイト季節反復: 単発遠征より、城ヶ島・三ツ石・鵜原など3サイトに絞り年4回ずつ訪問するほうが論文化可能な時系列データになる。
  5. 地元漁協/博物館と関係構築: 葉山しおさい博物館・東大三崎・真鶴遠藤貝類博物館とは早めに連絡を取り、標本同定依頼ルートを確保。
  6. しおさい博物館観察会/生命の星地球博物館イベントに参加: 学芸員の現地指導を受け、地元研究者ネットワークに入る最短ルート。
