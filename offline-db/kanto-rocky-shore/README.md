# 関東の磯・潮だまり マイクロハビタット総合リサーチデータセット

**生成日:** 2026-04-17 / **用途:** オフライン閲覧用リファレンス

## 横断的メモ

### 1. 分類群横断ニッチ軸
- **垂直分帯 (Z軸)**: 飛沫帯→潮上帯→潮間帯上中下→潮下帯。各帯で動物・植物が交代する (02章ASCII模式図)。
- **マイクロハビタット (X軸)**: 露岩/プール/オーバーハング/転石下/クレバス/藻場/CCA/砂礫/海綿マット/貝殻空隙 の 10 種。
- **波当たり勾配 (Y軸)**: 外洋暴浪 (外房・城ヶ島西) ↔ 内湾静穏 (東京湾・富津) で種組成が劇的に変わる。
- これら 3 軸で整理すると、関東磯の約 100 種はほぼ「排他的」なニッチ配列を形成 (05章)。

### 2. 関東磯の4つの生物地理ブロック
- **外房 (勝浦・鴨川・太海・守谷)**: 太平洋暴浪型、アラメ優占、黒潮直接影響
- **内房 (富津〜保田)**: 内湾静穏、カジメ+ワカメ、東京湾底棲種
- **三浦〜葉山〜真鶴 (相模湾北岸)**: 中庸、カジメ林豊富、博物館/臨海実験所密度最大
- **伊豆北東 (湯河原〜城ヶ崎)**: 亜熱帯要素混入、南方化フロンティア

### 3. 南方化フロンティア (07章)
- ガンガゼ *Diadema setosum* は 2025 年に福島県まで記録 — 従来「房総以南」の常識は崩壊中
- ヒョウモンダコも相模湾北部まで上昇 — 磯観察時の注意必要
- ソラスズメダイ・ミナミクロダイ・トラザメ も関東で繁殖確認
- 冬季最低水温 **15℃** を超えるとアラメ・カジメ休眠卵が失敗 → 磯焼け加速

### 4. 漁業権・法令 (06章)
- 2020年改正漁業法で **アワビ・ナマコ・シラスウナギは「特定水産動植物」指定** — 密漁は **3,000万円以下の罰金** (共同漁業権違反の100万円より重)
- サザエ・イセエビ・タコは共同漁業権種
- カニ網・忍者くまでは神奈川県で禁止漁具
- 富士箱根伊豆国立公園 (伊豆北東) / 南房総国定公園 / 真鶴半島自然公園 / 三浦半島県立自然公園 で採集規制要確認

### 5. 在野研究者の付加価値領域
- **都県RDB (千葉/神奈川/東京) の磯潮だまり種** は記載不足が多い — 定点モニタリング寄与可能
- 分類群横断の観察記録 (1地点 1日) は学術定点調査に比べ希少 — CSV + 写真で蓄積する価値
- 葉山しおさい博物館 / 真鶴遠藤貝類博物館 / 観音崎自然博物館 / 神奈川県立生命の星地球博物館 は在野協力を歓迎
