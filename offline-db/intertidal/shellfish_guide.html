1. 法的規制 - 全国共通のルール

最重要: 特定水産動植物は絶対禁止
2020年12月施行の改正漁業法により、以下の3種は漁業権や許可なく採捕すること自体が全面禁止です。 1匹でも採ると犯罪になります。「知らなかった」「自分で食べるだけ」は通用しません。
対象種罰則備考
アワビ（全種）3年以下の拘禁刑 or 3,000万円以下の罰金トコブシも含む
ナマコ（全種）同上マナマコ等すべて
シラスウナギ同上（2023年12月〜）ウナギの稚魚
アワビ - 特定水産動植物として全面採捕禁止 / Wikimedia Commons

運搬・保管・譲受・処分の媒介も同等の罰則。密漁品と知りながら購入した場合も処罰対象。

漁業権による保護

特定水産動植物以外にも、第一種共同漁業権で指定された水産物を漁業者以外が採ると漁業権侵害（100万円以下の罰金）になります。

サザエ / アサリ / ハマグリ / トコブシ / イセエビ / ウニ / タコ / ワカメ / ヒジキなど
※対象種は地域（漁業権の免許内容）によって異なります
サザエ - 多くの地域で漁業権対象 / Wikimedia Commons

漁業権の調べ方

  • 水産庁の海面漁業権情報: 各都道府県の漁業権一覧を確認可能
  • 各都道府県の水産課: 電話で問い合わせが最も確実
  • 現地の看板: 漁協が設置した禁止看板を必ず確認
  • 漁協に直接聞く: その海域の漁協に電話するのがベスト

大原則

3. 自由に採れる貝（漁業権対象外が多いもの）

注意
以下は「多くの地域で漁業権の対象になっていない」貝類ですが、 100%フリーではありません。地域によっては対象になっている場合があります。 必ず現地確認してください。
スガイ
Lunella coreensis
多くの地域でOK 美味

サザエの仲間。2cm程度の黒くて丸い巻貝。潮間帯の岩場にまとまって生息。味はサザエに似て美味。塩茹でが定番。

イシダタミガイ
Monodonta confusa
多くの地域でOK 出汁向き

3〜4cmの巻貝。石畳のような模様が特徴。潮だまりに群生。身は小さいが磯の香り豊か。塩茹でや炊き込みご飯のダシに。

タマキビ
Littorina brevicula
多くの地域でOK 珍味

1cm未満の極小巻貝。飛沫帯の岩に大量に付着。塩茹でして爪楊枝で身を出す。小さいが味は良い。

イボニシ
Thais clavigera
多くの地域でOK 濃厚

ゴツゴツした殻の肉食性巻貝。潮間帯のどこにでもいる。濃厚な旨味とダシが特徴。内臓には独特の苦味と辛味があり珍味として珍重。

カメノテ
Capitulum mitella
多くの地域でOK 絶品
カメノテ - 甲殻類の仲間 / Wikimedia Commons

厳密には貝ではなく甲殻類（エビ・カニの仲間）。岩の割れ目に群生。カニやエビに近い濃厚なダシと風味。スペイン料理「ペルセベス」の近縁種。塩茹でが絶品。

ヒザラガイ
Acanthopleura japonica 等
多くの地域でOK 通向き
ヒザラガイ（多板綱）/ Wikimedia Commons

8枚の殻板を持つ原始的な軟体動物。岩にぴったり張り付く。食べられるのは「足」の部分。アワビに似た食感。酢味噌（ぬた）が定番。ただし非常に硬いので長時間煮込むか低温調理が必要。

マツバガイ（カサガイ類）
Cellana nigrolineata 等
多くの地域でOK 美味
カサガイの仲間 / Wikimedia Commons

ペタッとした笠形で岩に張り付く。ナイフを横から素早く差し込んで剥がす。アワビに近い味わいで噛むと濃厚な海の風味。刺身・塩茹で・酒蒸し・バター焼きいずれも美味。

イソニナ
Japeuthria ferrea
多くの地域でOK バイ貝風

バイ貝（エゾバイ）の仲間。水中の石の裏に多い。危険を感じると自ら落下する性質あり。バイ貝に似た味で美味。

条件付きで採れる可能性がある貝

マテガイ
Solen strictus
地域による 美味
マテガイ / Wikimedia Commons

細長い二枚貝。砂浜の穴に塩を入れると飛び出してくる独特の採り方。漁業権の対象となっている地域あり。認可された潮干狩り場で楽しむのが安全。アサリより甘みが強く、バター焼き・酒蒸しが絶品。

シジミ
Corbicula japonica
地域による 出汁の王様

汽水域（海水と淡水が混じる場所）に生息。漁業権が設定されていない河川なら採取可能な場合がある。宍道湖・十三湖など有名産地は漁業権あり。味噌汁のダシは格別。

アサリ
Ruditapes philippinarum
認可場所のみ 万能
アサリ / Wikimedia Commons

多くの海域で漁業権対象。認可された潮干狩り場でのみ採取可能。勝手に採ると密漁。潮干狩り場では入場料を払い、ルール（サイズ制限・量の制限）を守ること。

カキ（マガキ）
Crassostrea gigas
地域による 絶品
マガキ（太平洋カキ）/ Wikimedia Commons

養殖が盛んで漁業権対象の地域が多い。漁業権区域外の岩場で天然物を採れる可能性はあるが、必ず事前確認が必要。ノロウイルス・貝毒のリスクが高いため、天然物の生食は危険。

4. 岩牡蠣（イワガキ）の扱い

岩牡蠣とは

マガキ（冬が旬）に対して、岩牡蠣は夏が旬の大型牡蠣。水深2〜10mの岩場に生息し、真牡蠣より大きく（10cm以上になる）、クリーミーで濃厚な味わい。

項目真牡蠣（マガキ）岩牡蠣（イワガキ）
旬11月〜3月（冬）6月〜9月（夏）
サイズ5〜10cm10〜20cm
味さっぱりクリーミー・濃厚
生息場所浅瀬〜干潟岩礁の深場
養殖主流天然物が多い

漁業権と天然岩牡蠣

岩牡蠣の採取は原則として不可と考えるべき
岩牡蠣は多くの地域で共同漁業権の対象になっています。 素潜りで採る行為は密漁として検挙される可能性が高いです。
  • 秋田県・山形県・新潟県: 天然岩牡蠣の産地だが漁業権で保護
  • 石川県・富山県: 同様に漁業権対象
  • 鳥取県・島根県: 夏の名物だが漁業者のみ採取可
  • 長崎県・大分県: 九十九島等で有名だが漁業権あり
天然岩牡蠣を合法的に楽しむ方法
  • 地元の直売所・市場で購入する
  • 漁師の案内付き体験ツアーに参加する
  • 岩牡蠣が名物の海鮮レストランで食べる

養殖岩牡蠣

近年は養殖技術が発達し、広島県・宮城県・三重県などで岩牡蠣の養殖が行われています。養殖物は安全管理されており、生食用としても流通しています。ふるさと納税の返礼品としても人気。

各貝の詳細な味の特徴

カメノテ ★★★★★

エビ・カニに近い甲殻類の旨味。塩茹でするだけで驚くほど濃厚なダシが出る。スペインでは高級食材「ペルセベス」として知られる。磯遊びで採れる食材としては間違いなく最高峰。

マツバガイ ★★★★★

アワビに近い本格的な味わい。噛むと濃厚な海の風味が広がる。刺身にすると透明感のある身でコリコリ食感。小型のアワビと言っても過言ではない。

マテガイ ★★★★★

アサリより甘みが強く、貝の旨味が濃縮された味。バター焼きにすると最高。採る楽しさも含めて潮干狩りの隠れた主役。

スガイ ★★★★

サザエの仲間だけあって味も良い。小さいが数で勝負。軽い苦味と磯の香りのバランスが絶妙。ワタの煮付けは通好み。

イソニナ ★★★★

バイ貝の仲間で味もバイ貝に似る。しっかりとした貝の味で、出汁も良く出る。煮付けにすると絶品。

イボニシ ★★★

濃厚な旨味。内臓の苦味・辛味は好みが分かれるが、珍味として好む人も多い。スープのベースにすると良いダシが出る。

ヒザラガイ ★★★

足の部分はアワビに似たコリコリ食感。ただし非常に硬い。長時間茹でるか低温調理で柔らかくなる。酢味噌で食べるのが伝統的。

6. 調理方法

下処理の基本

砂抜き（二枚貝の必須作業）

基本の砂抜き方法

  1. 貝を流水でよく洗い、殻の汚れを落とす
  2. 塩水を作る（水200mlに塩小さじ1 = 約3%濃度）
  3. 網付きバットに貝を重ならないように平らに並べる
  4. 貝の頭が少し出るくらいのひたひたの量で塩水を入れる
  5. アルミホイルや新聞紙で覆い、暗くする（竹串で空気穴）
  6. スーパー購入品: 1時間 / 潮干狩り採取品: 半日（6時間以上）
  7. 常温（20〜25度）が最適。冷蔵庫は冷たすぎて砂を吐かない
砂抜きのコツ
  • 貝が重なると下の貝が上の貝の砂を吸う = ジャリッの原因
  • 「なめてちょっとしょっぱい」程度の塩水がベスト
  • 暗い場所の方がよく砂を吐く
  • 砂抜き後に真水で洗うと貝が閉じて鮮度維持
巻貝の下処理

巻貝（スガイ、イシダタミ、イボニシ等）は砂抜き不要。以下の手順で:

  1. 流水でよく洗い、砂や泥を落とす
  2. 海水程度の塩水（約3%）で10分ほど浸ける
  3. 再度流水で洗う
  4. そのまま調理へ
カメノテの下処理
  1. 流水でよく洗い、付着物を取り除く
  2. 根元についた岩の破片などをナイフで除去
  3. そのまま茹でるか蒸す
  4. 食べ方: 外側の殻をむき、中の身（ピンク色）を食べる

調理法別ガイド

塩茹で / 塩ゆで（最もシンプル）

対象: スガイ、イシダタミ、タマキビ、カメノテ、イボニシ、イソニナ

  1. 鍋に海水程度の塩水を沸かす
  2. 貝を入れて5〜10分茹でる
  3. ザルにあげる
  4. 爪楊枝で身を取り出して食べる

最もシンプルで失敗がない調理法。磯遊び直後にそのまま楽しめる。

酒蒸し

対象: アサリ、ハマグリ、マテガイ、ホンビノス、カキ

  1. フライパンに貝を並べ、日本酒を回しかける（貝の1/3くらいの量）
  2. 蓋をして中火で加熱
  3. 貝の口が開いたらすぐに火を止める（加熱しすぎると硬くなる）
  4. お好みでバターや醤油を追加

バター焼き / バター醤油

対象: マテガイ、アサリ、マツバガイ、サザエ（購入品）

  1. フライパンにバターを溶かす
  2. 貝を投入し、全体にバターを絡める
  3. 醤油を回しかけ、蓋をして中火で蒸し焼き
  4. 香ばしい香りが立ったら完成

マテガイのバター醤油は特に絶品。甘みのある身にバターのコクが加わる。

味噌汁

対象: アサリ、シジミ、ハマグリ、小型巻貝各種

  1. 水から貝を入れて加熱（冷たい水から始める）
  2. 沸騰直前にアクを取る
  3. 貝の口が開いたら味噌を溶く
  4. 再沸騰させない（味噌の風味が飛ぶ）

シジミの味噌汁は二日酔いの定番。オルニチンが豊富。

炊き込みご飯

対象: 磯の小型巻貝全般、アサリ

  1. 貝を塩茹でして身を取り出す（茹で汁は取っておく）
  2. 米2合に茹で汁＋水で通常の水加減にする
  3. 醤油大さじ1、酒大さじ1、みりん少々を加える
  4. 通常通り炊飯
  5. 炊き上がったら身を戻して混ぜる

磯の香りと貝の甘みが米に染み込んで絶品。

ぬた（酢味噌和え）

対象: ヒザラガイ、カサガイ類

  1. 貝を3時間ほどしっかり茹でる（短いと硬い）
  2. 殻から足の部分を取り出す
  3. 薄切りにする
  4. 白味噌2:酢1:砂糖少々の酢味噌で和える

7. 危険な貝 - 絶対に触るな・食べるな

イモガイ類 - 刺傷による致死毒

イモガイは人を殺せる猛毒を持つ
1個体に約30人分の致死量の毒を持ち、刺されると呼吸麻痺で死亡する可能性があります。 特にアンボイナガイ（ハブガイ）は最も危険で、沖縄では死亡例が複数報告されています。 有効な抗毒素はありません。
イモガイの一種（タガヤサンミナシ）- 美しいが猛毒 / Wikimedia Commons
イモガイ類（Conus属）

特徴: 美しい模様の円錐形の巻貝。芋（サツマイモ）に似た形からイモガイ。

毒の仕組み: 歯舌が変化した「銛（もり）」で刺し、神経毒を注入。刺した瞬間は痛みを感じないことが多い。

症状: しびれ → 激痛 → 腫れ → めまい・嘔吐 → 全身麻痺 → 呼吸不全（重症時）

被害地域: 主に琉球列島（沖縄〜奄美）。温暖化で分布北上中。紀伊半島・四国南部でも注意。

対策

  • 綺麗な貝殻を絶対に素手で拾わない
  • 水中で貝を見つけても触らない
  • 刺された場合は直ちに救急車（119番）
  • 毒を吸い出す・縛るなどの民間療法は無効

食べると危険な部位

中腸腺（肝膵臓）- 通称「ウロ」

巻貝の渦巻き部分の奥にある暗緑色の内臓。以下の理由で注意が必要:

  • 重金属の蓄積: カドミウム、鉛、ヒ素などが濃縮される
  • 貝毒の蓄積: 二枚貝だけでなく巻貝でも蓄積される場合あり
  • テトラミン: ツブ貝（エゾバイ科）の唾液腺に含まれる毒。食後30分〜1時間で頭痛・めまい・酩酊感。加熱しても分解されない
エゾバイ科（ツブ貝）の唾液腺は必ず除去
イソニナもエゾバイ科の仲間。大量に食べる場合は内臓を除去してから調理するのが安全。

寄生虫リスク

アニサキス

主に魚介類の内臓・筋肉に寄生する線虫。貝類への寄生は比較的少ないが、二枚貝を中間宿主とするケースがある。

  • 予防: 加熱（60度1分以上）or 冷凍（-20度24時間以上）
  • 磯の巻貝は通常しっかり加熱するため問題になりにくい
  • 生食する場合（マツバガイの刺身等）は十分注意

住血吸虫・肝吸虫

淡水貝（タニシ等）を生食すると感染リスク。淡水貝は必ず十分に加熱すること。

8. 貝毒 - 見えない脅威

貝毒は加熱しても消えない
貝毒の毒成分は熱に強く、煮ても焼いても毒性は弱くなりません。 また、外見では毒の有無を判別できません。
種類症状発症時間重症度
麻痺性貝毒 口唇・舌・顔面のしびれ → 四肢麻痺 → 呼吸麻痺 食後30分 致死の可能性あり
下痢性貝毒 水様下痢・嘔吐・吐き気・腹痛 食後30分〜数時間 通常3日以内に回復

貝毒が溜まりやすい貝

  • 二枚貝全般: アサリ、ムラサキイガイ（ムール貝）、ホタテ、カキ、アカガイ等
  • プランクトンを濾過して食べるため、有毒プランクトンの毒素が蓄積する
  • 巻貝は捕食方法が異なるため比較的リスクは低いが、ゼロではない

安全対策

貝毒から身を守るために
  • 出荷されている貝は安全: 生産海域では定期的に貝毒検査が行われている
  • 自分で採った二枚貝は要注意: 検査されていない
  • 各都道府県の「貝毒情報」を事前にチェック
  • 赤潮が発生している海域の貝は絶対に食べない
  • 春〜夏（4月〜9月）は貝毒の発生が多い時期
  • 異変を感じたら直ちに医療機関へ（食べた貝を持参）
巻貝は比較的安全
磯遊びで採れる小型巻貝（スガイ、イシダタミ、イボニシ等）は二枚貝と異なり プランクトンを濾過しないため、貝毒のリスクは相対的に低いです。 ただし、ヒザラガイで貝毒による食中毒の報告例はあります。

9. 磯遊び 実践的な採集ガイド

潮汐の見方

干潮を狙え

磯遊び・潮干狩りは干潮の2時間前〜干潮時がベストタイム。

  • 大潮（新月・満月の前後）: 潮の引きが最も大きく、普段水没している場所まで露出。最高の採集チャンス
  • 中潮: まずまず。大潮の前後数日
  • 小潮: 潮の引きが小さい。採集には不向き
  • 長潮・若潮: 最も引きが小さい
おすすめアプリ
「潮汐なび」「しおさい」「タイドグラフBI」などのアプリで干潮時刻を確認。 春の大潮（3〜5月）は昼間に大きく潮が引くため最も快適に採集できます。

採集のコツ

磯の巻貝

  • 潮だまり（タイドプール）を重点的に探す
  • 石の裏側、岩の割れ目をチェック
  • カメノテは波が強く当たる岩の割れ目に群生
  • カサガイ類は横からすばやくヘラを入れる（一度警戒されると剥がせない）
  • スガイ・イシダタミは岩の上面にまとまっている

干潟の二枚貝

  • アサリは砂の表面にある小さな穴（吸水管の穴）を探す
  • 穴が2つ並んでいたらアサリがいる確率大
  • 10cmの深さまで掘っていなければ場所を変える
  • マテガイは穴に塩を入れて3〜5秒待つと飛び出してくる
  • 飛び出したら素早く掴んで引き抜く（逃げるのが速い）

持ち帰り方

  1. 採った貝を真水で軽く洗う（貝が閉じて鮮度維持）
  2. 濡れた新聞紙で包む or そのままクーラーボックスへ
  3. 海水に浸けたまま持ち帰らない（酸欠で死ぬ）
  4. 保冷剤で冷やす（直接貝に触れないよう新聞紙で間を作る）
  5. 帰宅後すぐに砂抜き → 調理
  6. 翌日まで持ち越さないのが理想
採りすぎ注意
その日に食べきれる量だけ採り、残りは必ず元の場所に戻しましょう。 磯の生態系は繊細です。小さな個体は次世代のために残してください。

マナーと注意事項

  • 石をひっくり返したら必ず元に戻す（下の生物が死ぬ）
  • ゴミは持ち帰る
  • 漁業者の作業を邪魔しない
  • 子供から目を離さない（波に注意）
  • 単独行動は避ける（転倒時に助けを呼べない）
  • 天候の急変に注意（雷、高波）
  • 帰りの潮の満ち込みに注意（取り残されないよう）
