# オオクワガタ (Dorcus hopei binodulosus) 群馬県記録まとめ

- 調査日: 2026-04-20
- 学名: *Dorcus hopei binodulosus* Waterhouse, 1874
- 和名: オオクワガタ
- 調査目的: 群馬県における採集記録・生息情報の集約
- 注意: 密猟対策のため、具体地点(詳細な林分・ポイント)は市町村レベルまでに留めている

---

## 1. 総括 (Executive Summary)

群馬県はオオクワガタの分布北東縁域にあたる関東内陸県であり、「生息はしているはずだが正式な採集記録がほとんど公表されていない」という典型的な "情報欠落型産地" と評価できる。Web調査の範囲では、以下が主要所見。

- **ネット上で確認できる「信頼できる群馬県産オオクワガタ採集記録」はほぼ皆無**。採集者は乱獲防止のため地名を伏せる (「某所」「北部」等) 傾向が強い。
- 関東地方の産地総覧サイト (Dorcus Navi等) は「**群馬県では採集記録はないが、生息していることは間違いない**」という定型文を採用している [未検証]。
- 群馬県北部 (沼田市以北・片品・みなかみ等) のブナ帯では **ヒメオオクワガタ (D. montivagus)** の採集記が多数あるが、これはオオクワガタとは別種で、標高帯 (1,000 m 以上のブナ-ミズナラ帯) と樹種 (ヤナギ・ミズナラ類) が異なる。
- 群馬県立ぐんま昆虫の森 (桐生市新里町) が県内昆虫相の中核施設だが、Web上にはオオクワガタ標本・県内採集記録の公開データベースは見当たらない。
- 河川敷 (利根川・渡良瀬川水系) でのクワガタ採集記は存在するが、**採集種はヒラタクワガタ・コクワガタ・ノコギリクワガタが大半**で、オオクワガタを明記した記事は本調査では発見できなかった。

## 2. 地域別整理

### 2-1. 利根川本流流域 (前橋・伊勢崎・太田・玉村ほか)

| 項目 | 内容 |
|---|---|
| 採集地 | 利根川河川敷 (具体市町村は非開示、沼田〜伊勢崎〜太田間と推定) |
| 樹種 | ヤナギ類 (タチヤナギ等)、オニグルミ、ニレ (ハルニレ) |
| 採集種 | ヒラタクワガタ主体、ノコギリクワガタ、コクワガタ、カブトムシ。**オオクワガタは明記なし** |
| 時期 | 8月中旬 (月夜野きのこ園 2016年記録) |
| 個体数 | 21頭 (5種合計、2016年8月) |
| 備考 | 一般にハルニレのウロには「ヒラタ、時にオオクワガタも入る」とされるが、群馬県利根川筋での具体記録は未確認 [未検証] |
| 出典 | [月夜野きのこ園 2016年 河川敷採集記](https://kuwakabu.tsukiyono.co.jp/2016/09/06/) / [河川敷クワガタ採集 (Ameblo)](https://ameblo.jp/takumabear/entry-12389463158.html) |

### 減少要因 (全国共通)
1. **乱獲** (1980年代後半〜1990年代の「オオクワブーム」で顕著)
2. **里山雑木林の消失・薪炭利用の停止によるクヌギ・コナラの老齢化不足**
3. **河川敷の河畔林 (ヤナギ・ニレ) の減少・治水工事**
4. **飼育個体 (外国産D. hopei hopei含む) の逃亡・放虫による遺伝的攪乱リスク**

## 6. 限界・今後の調査課題

1. **群馬県RDL 2022 PDF本文のカテゴリ確認** — WebFetchではPDFバイナリ解析に失敗。ローカル `curl` + `pdftotext` で再取得要。
2. **ぐんま昆虫の森 所蔵標本ラベルデータ** — Web未公開の可能性が高い。現地または問い合わせベースで確認するしかない。
3. **同好会誌**: 群馬昆虫学会・上毛昆虫同好会の会報 (『ぐんま昆虫たより』等 [未検証]) に県内採集記録がある可能性。国会図書館サーチ・J-STAGEで要追加検索。
4. **S-UMSUM (標本DB)・GBIF日本インポート** — `Dorcus hopei binodulosus` × `stateProvince=Gunma` で直接クエリ可能。
5. **戦前〜1970年代の古い採集記録** — 「昔は記録があったが最近は聞かない」型のパターン。旧文献 (月刊むし、昆虫、Gekkan Mushi 1986年オオクワガタ特集号等) のバックナンバー要調査。
