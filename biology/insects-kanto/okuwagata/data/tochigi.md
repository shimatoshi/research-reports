# オオクワガタ (Dorcus hopei binodulosus) 栃木県 採集・生息記録

調査日: 2026-04-20
調査者: kobayashit
対象: *Dorcus hopei binodulosus* Waterhouse, 1874 (オオクワガタ本州亜種)
調査範囲: Web上で公開されている個人ブログ・同好会情報・自治体レッドデータ等

注意: 密猟対策のため、詳細な地点情報 (GPS、具体的な林班等) は収集対象から除外し、市町村・流域レベルまでで集約する。

---

## 1. サマリ

- 栃木県はオオクワガタの東日本産地として **塩谷郡・那須・足利・塩原** などが採集コミュニティで知られる ([Dorcus Navi 関東](https://dorcusnavi.com/habits/growing-area/kanto/), [Yahoo!知恵袋](https://detail.chiebukuro.yahoo.co.jp/qa/question_detail/q1244607847))。
- 県北部ではブナ・ミズナラ帯を経由して **福島県南会津郡と地続き** で採集記録がある ([Dorcus Navi](https://dorcusnavi.com/habits/growing-area/kanto/))。
- ブログ等では **全地域に分布するが個体数はかなり少ない** とまとめられている ([Kuwashinブログ Part1](https://kuwashin27.exblog.jp/24067596/))。
- 環境省 RL では **絶滅危惧II類 (VU)** (2007年改訂で準絶滅危惧から引き上げ) ([Wikipedia](https://ja.wikipedia.org/wiki/%E3%82%AA%E3%82%AA%E3%82%AF%E3%83%AF%E3%82%AC%E3%82%BF))。
- 栃木県版レッドリスト (2023年第4次) の個別カテゴリは WEB 上で確認できず[未検証]。那須塩原市 RL 2023 には **ヒメオオ/コルリ/ルリ/ヒラタ** 等は掲載されているが、オオクワ *D. hopei* は未掲載 ([那須塩原市 RL 2023 PDF](https://www.city.nasushiobara.tochigi.jp/material/files/group/13/RL2023.pdf))。

## 2. 地域別記録

### 2.1 那須・塩原地域 (那須塩原市、那須町、大田原市周辺)

| 項目 | 内容 |
|---|---|
| 産地知名度 | 「栃木は塩谷や那須あたりいます」と採集者コミュニティで繰り返し言及 [未検証・具体地不明] |
| 樹種 | クヌギ・コナラ主体の里山、標高の上がるところではミズナラ帯 |
| 採集方法 | 街灯採集が「茨城より栃木の方が捕れる」と記述 |
| 時期 | 7月上旬〜8月下旬、新月前後が効果的 |
| 個体数 | 記録はあるが年数個体レベル、個体数は少ない |

情報源:
- [Yahoo!知恵袋 (栃木・茨城のオオクワ樹液ポイント)](https://detail.chiebukuro.yahoo.co.jp/qa/question_detail/q1244607847)
- [那須塩原市レッドリスト 2023 (PDF)](https://www.city.nasushiobara.tochigi.jp/material/files/group/13/RL2023.pdf) — D. hopei の掲載は確認できず (掲載は D. montivagus ヒメオオ、 Platycerus 2種、 D. platymelus pilifer ヒラタ等)
- [那須昆虫ワールド (那須高原)](https://www.nasukon.com/liveinsects/) — 地域の昆虫相参照

### 2.2 日光・奥日光・栗山・川俣温泉

| 項目 | 内容 |
|---|---|
| 地域 | 旧栗山村 (現日光市)、奥日光川俣温泉周辺 |
| 環境 | ブナ・ミズナラ帯、標高1,000m以上の山地域 |
| 記録評価 | あるブロガーが「ブナ・ミズナラ帯が保全されたポイントとして申し分ないが、3年間偵察の結果、甲虫の存在は薄くオオクワ採集は難しい」との結論 [未検証] |
| 同所的種 | ヒメオオクワガタ・ルリ/コルリクワガタは豊富 |
| 採集時期 | 夏期 (7–8月) のトラッキング採集 |

### 2.5 渡良瀬遊水地・鬼怒川流域 [未検証]

- 生態的にヤナギ樹液にも集まるため可能性はあるが、県境部の渡良瀬遊水地 (栃木・群馬・埼玉・茨城) では群馬・埼玉側の情報が先行。
- 鬼怒川上流 (日光市・塩谷) は上流部=山地帯に区分される。

情報源:

## 3. 環境・生態サマリ

| 項目 | 内容 | 備考 |
|---|---|---|
| 主要樹種 | クヌギ・コナラ・ミズナラ・アベマキ・カシ類・ニレ類・ヤナギ類 | [Wikipedia](https://ja.wikipedia.org/wiki/%E3%82%AA%E3%82%AA%E3%82%AF%E3%83%AF%E3%82%AC%E3%82%BF) |

### 減少理由 (一般論、栃木県にも該当するとみられる)

- 乱獲 (高値取引・ブリード目的) ([Wikipedia](https://ja.wikipedia.org/wiki/%E3%82%AA%E3%82%AA%E3%82%AF%E3%83%AF%E3%82%AC%E3%82%BF))
- 里山・丘陵地の開発、放置雑木林の遷移進行 (ナラ枯れ含む)
- 大径木 (樹洞を形成する老齢木) の伐採・消失
- 河畔ヤナギ林の護岸・河川改修による減少 (栃木でも鬼怒川・渡良瀬で懸念) [未検証]
