# 印旛オオクワガタ伝聞の出典トレース

調査日: 2026-04-27
調査目的: 「印旛地域でオオクワガタ (*Dorcus hopei binodulosus*) が採集できる」という話の初出・一次情報源を特定する

> 注 (chiba.md と同じ方針): 採集ピンポイント (林道名・神社名・GPS等) は記載しない。市町村名までに丸める。

---

## 調査結果サマリー

- **最古の言及 (商業/Web)**: あいあん (iron) 「オオクワガタの有名産地」 (くわばか2000秋号, 2000年頃, asahi-net.or.jp/~id8k-sgn/) — ただし「東関東」(千葉・茨城) としか書かれず、印旛 / 印西 / 横芝 等の固有地名は登場しない
- **最古の固有地名併記 (商業/Web)**: 天羽ドルクス「千葉県産オオクワガタの産地別飼育」(2008-05-31, amahadorcus.bosolife.net/gallery-chiba.htm) で **印旛地域・印西市・白井市・横芝光町・松尾町・勝浦市・市原市** を地図表示
- **最古の流通ラベル**: クワガタ・カブト専門店 ＭＧＭ で「**千葉県印西市印旛産** 材割成虫採集 ♂66ミリ ♀42ミリ ペア」が販売 (Wayback 確認の最古スナップショット 2021-05-14, 流通開始時期は不明だがそれ以前)
- **最古の学術一次記録 (印旛地域)**: 鈴木智史・城田義友 (2009)「成田市でオオクワガタを採集」『房総の昆虫』No.43: 32 — 2008-11-30 ♂、2008-12-14 ♀+幼虫 (材割採集、コナラ)
- **一次情報源の有無**: **部分的にあり**。成田市 (印旛地域北端) からは2008年の正式記録が同好会誌に掲載されている。一方、**印旛村・印西市・白井市の野外オオクワガタは、1977–1985年の西印旛沼周辺甲虫調査 (鈴木 2009, 房総の昆虫 No.43) でクワガタムシ科3種 (ノコギリ・コクワ・ヒラタ) が確認されたものの、オオクワガタは記録されていない**
- **推定される伝聞の伝播経路**:
  1. 1990年代後半〜2000年前後の「クワガタブーム / 黒いダイヤ」流行
  2. BE-KUWA No.8『日本のオオクワガタ大特集!!』(2003年9月発行, むし社) で全国250採集ポイントを掲載 — Dorcus Navi の関東地方解説でも「**千葉では印旛郡・山武郡・市原市・夷隅郡・勝浦市**」とされ、出典は同号
  3. これに前後して、千葉の昆虫店 (例: 習志野市の習クワ) が「白井産」「千葉県産」ラベルで個体流通
  4. ＭＧＭ等が「千葉県印西市印旛産」ラベルで材割成虫を販売
  5. 2008年に 天羽ドルクス が「千葉県産オオクワガタ産地別飼育」ページを公開 (印旛・印西・白井・横芝・松尾・勝浦) — 千葉県外のクワガタファンが見る一番分かりやすい産地マップとなり、Yahoo知恵袋 (2011-2012) や個人ブログを通じて拡散
  6. 「印旛沼周辺は雑木林が残るオオクワ生息地」という共通認識が定着するが、**学術同好会誌レベルでの印旛村・印西市・白井市での野外個体記録は本調査時点では未発見**

## 一次資料 (博物館・学術) の照合結果

### 千葉県昆虫談話会『房総の昆虫』(1989年〜現在 No.73 まで発行)

千葉県の昆虫相を体系的に記録する事実上の最重要同好会誌 (NDLサーチでも「千葉県の昆虫相を研究する上で欠かすことの出来ない文献」と評価)。

- **印旛地域で確認できるオオクワガタ唯一の同好会誌掲載記録**:
  - 鈴木智史・城田義友 (2009)「成田市でオオクワガタを採集」『房総の昆虫』No.43: 32. 採集データ:
    - 1♂+1幼虫, 成田市北部, 2008-11-30, 鈴木智史採集 (1♂は鈴木が保管、幼虫は織戸昌則氏が飼育)
    - 1♀+1幼虫, 成田市北部, 2008-12-14, 城田義友採集・保管
    - 採集方法: コナラの朽ちた部分の材割。マイマイカブリ採集を目的に訪れた際の偶然の発見
- **印旛村・印西市・白井市・本埜村の野外オオクワガタ記録**: **本調査時点 (2026-04-27) で『房総の昆虫』No.43〜No.73 の目次および取得済PDF (No.43, No.52, No.58) からは確認できず**
- 鈴木 勝氏の長期系統調査 (1977-1985 + 続報シリーズ I-III および補遺・追加 1-4) において**西印旛沼周辺地域 (印旛村・印西市・本埜村・佐倉市)からのクワガタムシ科は ノコギリ・コクワ・ヒラタの3種に留まる**
- 2009年12月以降、佐倉市・印旛沼関連の記事は多数あるが、いずれもオオクワガタには触れない
- ただし、No.43~73すべてを精査したわけではなく、目次からの探索のみであるため、見落としの可能性あり

## 結論と未解決の問い

### 結論 (要約)

1. **「印旛地域でオオクワガタが採集できる」という伝聞の構造**:
   - 1990年代後半〜2000年前後の **クワガタブーム / 黒いダイヤ** 時代に「千葉でも採れる」という話が一般化
   - **2003年9月発行の BE-KUWA No.8 が250採集ポイント情報を集約・公開**し、「印旛郡・山武郡・市原・夷隅・勝浦」という千葉県内産地名がブランド化
   - 2008年の天羽ドルクス産地別マップで地域名 (印旛・印西・白井・横芝・松尾) が固定化
   - 2010年代に Yahoo知恵袋・個人ブログ・昆虫店ラベル (ＭＧＭの「印西市印旛産」、習クワの「白井産」等) によって伝聞ロンダリングが進行

2. **学術一次記録の現状**:
   - **印旛地域で唯一公的に同定・公表された個体**: 鈴木智史・城田義友 (2009)『房総の昆虫』No.43: 32 の **成田市** 2♂2♀ (2008年採集、コナラ材割)
   - 西印旛沼周辺地域 (印旛村・印西市・本埜村・佐倉市) の8年間の系統調査 (鈴木 勝 2009; 続報 2013-2018) では**オオクワガタは1個体も記録されていない**
   - 白井市・印西市・印旛村の野外個体記録は同好会誌・図鑑では未確認

3. **「伝聞=採集できる」と「学術記録=採れない」のギャップ**:
   - 印旛沼周辺の二次林環境ではオオクワガタは**極めて稀**で、「居なくはないが採れない」という認識が現実に近い
   - 流通している「印西市印旛産」は出元の信頼性が確認できない (採集者・採集年・標本番号いずれも不明)
