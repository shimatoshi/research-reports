# オオクワガタ (Dorcus hopei binodulosus) 埼玉県 採集・生息情報

**調査日:** 2026-04-20
**対象種:** オオクワガタ *Dorcus hopei binodulosus*
**対象範囲:** 埼玉県全域 (秩父・奥武蔵・比企丘陵・荒川流域)

> **注意:** 密猟対策のため具体的地点 (GPS座標・林道名・沢名・ポイント木) は記載しない。市町村レベルまでに留める。

## 1. サマリー

- 埼玉県は **戦前は東京23区近郊でも記録があった本種の分布域**だが、現在は生息地が極めて限られる。
- 現行記録の中心は **秩父郡 (秩父地域)**。比企丘陵・荒川中流域・奥武蔵では確実な近年記録が Web 上で確認できない。
- 埼玉県レッドデータブック動物編 2018 (第4版) では **絶滅危惧IA類 (CR)** に指定され、県内では最も危機的カテゴリ。
- 本調査で Web 上から抽出できた「具体的な近年採集記録」は **2件** (いずれも市町村名非公表・県内とだけ明記されたブログ記録)。

## 3. 現状の個体群評価

### 3.1 レッドデータ

| 指定 | カテゴリ | 出典 |
|---|---|---|
| **埼玉県レッドデータブック動物編 2018 (第4版)** | **絶滅危惧IA類 (CR)** | jpnrdb.napzak.jp (RDB検索システム転載) |
| 環境省レッドリスト 2020 | 絶滅危惧II類 (VU) | Wikipedia/環境省 |

- 埼玉県内では県レベルで**最も絶滅リスクの高いカテゴリ (CR)** に指定。国カテゴリ (VU) より1ランク厳しい。

### 3.2 減少理由 (一般論・埼玉に適用)

1. **乱獲**: 市場価値が高く、秩父産個体を狙ったプレッシャーが継続。
2. **里山管理放棄**: 薪炭林としてのクヌギ伐採サイクル消失 → 台場クヌギ (萌芽更新された老大木) の減少。
3. **宅地・ゴルフ場開発**: 比企丘陵・奥武蔵低山では開発影響。
4. **大径木の喪失**: 樹洞を持つ老齢樹減少で成虫の越冬・潜伏場所が不足。
5. **河川改修**: 荒川流域のヤナギ大木消失。

### 3.3 系統的位置 (参考)

- 戦前の関東平野での分布を考慮すると、埼玉県個体群は **関東在来系統**の残存可能性あり。
- 「関東産オオクワには特段の形態的特徴はない」との指摘 (Dorcus Navi)。
- 近年は飼育個体の野外放逐による **遺伝的撹乱**の懸念が別途存在 (全国共通課題)。
