# オオクワガタ 関東採集記録データセット

**Dorcus hopei binodulosus** — 関東7都県における採集・生息記録のオフライン資料集。

- 作成日: 2026-04-20
- 目的: ネット遮断環境でも参照できる一次整理済みデータ。論文・調査の素材として使う前提。
- 収集方法: Web調査 (WebSearch/WebFetch) を並列で実行しマークダウン化。一次資料の確認は別途必要。

## ⚠ 取り扱い注意

- 採集地点は**市町村レベルまで**。詳細地点・GPS は記載しない (密猟防止)。
- 未確認・伝聞情報には `[未検証]` タグ付き。論文等で使う前に原典確認のこと。
- 全都県で環境省VUまたは各県RDB掲載種。採集・公開には倫理的配慮を。

## データの傾向 (速報)

- **公開されている具体的採集記録は茨城・千葉・東京に集中** (各9〜14件)。
- **群馬・栃木は「生息確実だが公表記録なし」**。採集者が密猟警戒で伏せている傾向。
- **埼玉・神奈川は県RDBで最重要保護クラス**。近年記録は各1件のみ。
- 共通する好環境: クヌギ・コナラ樹液、河畔ヤナギ林、標高200〜800mの二次林。
- 減少要因: 里山放棄、薪炭林伐採停止、乱獲、生息地分断 (共通)。
