# ムサシトミヨ調査プロジェクト — OsmAnd レイヤー

白子川源流(大泉井頭公園)を主候補としたムサシトミヨ再発見調査用の地図レイヤー。
調査経緯と各地点の確定情報をGPX waypointの`<desc>`に紐付けて保存している。

## ファイル

| ファイル | 用途 |
|---|---|
| `musashitomyo-shirakogawa.gpx` | OsmAnd読込用 (waypoints 14点) |
| `build_gpx.py` | GPX生成スクリプト。データ追加時はこれを編集して再実行 |
| `README.md` | この文書 |

## OsmAnd への読み込み手順

### Android (OsmAnd公式アプリ)
1. GPXファイルをスマホに転送 (USB / Googleドライブ / メール添付 等)
2. OsmAndを起動 → メニュー → **「軌跡」** または **「インポート」**
3. `musashitomyo-shirakogawa.gpx` を選択
4. 表示設定で `<extensions>` の色情報が反映される (赤=最優先 / 橙=歴史 / 紫=参考 / 青=連携 / 緑=観察点)

### 直接配置 (上級)
OsmAndのトラックフォルダに直接置く方法:
```
/storage/emulated/0/Android/data/net.osmand.plus/files/tracks/
```
配置後、OsmAnd内で「軌跡」→「読み込み済み」から有効化。

### iOS (OsmAnd Maps)
1. ファイルアプリでGPXを保存
2. 共有 → OsmAnd Maps を選択
3. 「お気に入り」または「軌跡」として読み込まれる

## 調査経緯のサマリ

1. 「印旛沼で消滅したムサシトミヨを探す」が出発点
2. 一次資料調査 → **千葉県分布説は学術根拠なし** (Wikipedia citogenesis 由来) と判明
3. 学術一次出典での過去分布は **東京3地点 + 埼玉3地点** のみ
4. 残存集団復活の現実的候補は武蔵野台地ハケ湧水ネットワーク
5. 23区内アクセスと制度・人的密度を重視し **白子川 (大泉井頭公園源流)** を主候補に絞る
6. PDF原典 + nitter経由SNS情報で **ホトケドジョウ繁殖中** + **湧水・水草・低水温** の三条件揃いを確認
7. 「白子川源流・水辺の会」(@sugalohas, 毎月第四日曜定例活動) という強力な連携先を発見

## 更新方法

新しい地点・情報を追加する場合:

```bash
cd /home/kobayashi-takeru/musashitomyo-survey
# build_gpx.py の WAYPOINTS リストに dict を追加
python3 build_gpx.py
# 出力されたGPXをスマホに再転送
```

座標が不確実な場合は `desc` 末尾に「※ 座標は概略」と明記する慣習にしている。
