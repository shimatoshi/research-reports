## 凡例

- **D1**: 第1背鰭棘条数, **D2**: 第2背鰭条数, **A**: 臀鰭条数
- **P**: 胸鰭軟条数, **LS**: 縦列鱗数, **PDS**: 背鰭前方鱗数
- **TS**: 横列鱗数, **V**: 脊椎骨数
- **POC**: 後眼肩甲管, **POP**: 前鰓蓋管
- **SDP**: 第1背鰭起点−胸鰭基底上端間の鱗数

### シマヨシノボリ *Ctenogobius katonis* Tanaka, 1908
**出典**: Suzuki & Chen (2011) — タイプ標本再精査

| 形質 | 値 |
|------|---|
| D1 | VI |
| D2 | I,8 (9: 1個体) |
| A | I,8 |
| P | 19 (20: 1個体, 21: 1個体) |
| V | 10+16=26 |
| LS | 33 (32: 3個体) |
| TS | 9 (10: 2個体) |
| PDS | 13 (9: 1個体, 15: 1個体) |
| POC | あり (B'・C・D(S)・E・F・H') |
| POP | あり (M'・N・O') |

- **D1形状**: 第2棘最長、伸長せず三角形、倒しても第2背鰭に達しない
- **体色**: クリーム色と飴色のまだら、背側に6鞍状斑
- **頬部**: パラタイプに数本のミミズ状斑、尾鰭基底にカイザル髭状斑
- **胸鰭基部**: 瞳径大の飴色菱形斑あり、胸鰭前部に飴色三日月斑
- **尾鰭**: 全体に黄土色横点列
- **備考**: sp. CBに一致（明仁ほか, 2000; 鈴木・渋川, 2004）。*R. nagoyae*のシノニムの可能性あり

- **備考**: sp. OR偽橙色型に一致（明仁ほか, 2000; 鈴木ほか, 2010）。タイプ産地：東京港区黒田侯爵邸庭園内の池

### ルリヨシノボリ *R. mizunoi* Suzuki, Shibukawa & Yano, 2017

| D1 | V–VI |
| D2 | I,8 (I,9: 1個体) |
| A | I,8 (I,9: 1個体) |
| P | 18–20 |

| TS | 9–10 |

| SDP | 9–12 |

**出典**: Suzuki & Mukai (2019)

| 形質 | 値 |
|------|---|
| D1 | VI |
| D2 | I,8 |

| PDS | 3–15 |
| SDP | 6–7 |

- **D1形状♂**: 低い台形 or 将棋駒型、**第3棘最長**、倒しても第2背鰭起部に達しない

- **体長**: 最大39.5 mm SL — **小型矮化型**
- **生活史**: 陸封型（池沼・湿地・用水路の泥底+水草環境）
- **分布**: 東海地方（愛知・三重・岐阜・静岡）

- **生活史**: 陸封型（池沼・湿地・用水路）
- **分布**: 瀬戸内海・大阪湾・紀伊水道沿岸の内陸水域、東海地方にも（移入の可能性）

### ヤイマヒラヨシノボリ *R. yaima* Suzuki, 2020
**出典**: Suzuki (2020)

| 形質 | 値 |
|------|---|
| D1 | VI |
| D2 | I,7–9 (モードI,8) |
| A | I,7–8 (モードI,8) |
| P | 19–22 |
| V | 10+16=26 |
| LS | **40–43** — **属内最多** |
| TS | 10–13 |
| PDS | 0–17 |
| SDP | 9–12 |
| POC | あり (K', L') |
| POP | あり (M', N, O') |

- **頭部**: **扁平**（最大頭高/最大頭幅 = 65.5–71.6%）— 急流適応
- **体型**: 体と尾柄が**細長い**（体高: 13.8–14.5% SL）
- **D1形状♂**: 低い台形、D2よりわずかに高い程度、**倒しても第2背鰭起部に達しない**
- **第5腹鰭軟条**: **5分枝** — yonezawaiとの最重要区別（4分枝）
- **胸鰭基部**: **裸出**（鱗なし）、くすんだ橙色の楕円形斑
- **腹鰭前方**: 裸出
- **体色♂**: 側頭部に2本の橙色縦線（**第2背鰭起部下まで**伸びる、yonezawaiより長い）
- **尾鰭♂**: 4本の明るい橙色点の縦列
- **尾鰭基部♀**: 1対の垂直に並んだ丸形/長方形黒色斑
- **体長**: 最大66.0 mm SL
- **分布**: 八重山諸島（石垣島・西表島）の山地渓流上流部

### ケンムンヒラヨシノボリ *R. yonezawai* Suzuki, 2020
**出典**: Suzuki (2020)

| 形質 | 値 |
|------|---|
| D1 | VI |
| D2 | I,8 |
| A | I,8 |
| P | 19–21 |
| V | 10+16=26 |
| LS | 35–39 |
| TS | 9–12 |
| PDS | 2–14 |
| SDP | 8–10 |
| POC | あり (K', L') |
| POP | あり (M', N, O') |

- **D1形状♂**: **高い鎌型（falcate）**、D2よりはるかに高い、倒すと第2背鰭第1–4分枝軟条間に達する
- **第5腹鰭軟条**: **4分枝** — yaimaとの最重要区別（5分枝）
- **胸鰭基部**: 裸出、明瞭な**黒色の楕円形斑** — yaimaは橙色
- **体色**: 側頭部に2本の深赤色縦線（**第1背鰭下まで**、yaimaより短い）
- **尾鰭♂**: 6–8本の深赤色/赤橙色の**縦線**

- **D1**: 暗色斑なし（明示的に記載）
- **体長**: 最大75.2 mm SL
- **生活史**: 両側回遊型（Takagi et al. 2015）、沖縄島のため池で陸封も
- **分布**: 琉球列島（種子島・屋久島・奄美大島・沖縄島）の山地渓流上流部

## 同定フローチャート（原記載準拠）

```
Step 1: 脊椎骨数
├── V = 26–28 (モード27) → パイヌキバラヨシノボリ群 (R. aonumai) → Step 6へ
└── V = 26 → Step 2へ

Step 2: 縦列鱗数
├── LS = 40–43 → ヤイマヒラヨシノボリ (R. yaima) [頭部扁平、D1低い]
├── LS = 35–39 → ケンムンヒラヨシノボリ (R. yonezawai) [D1♂高い鎌型]
└── LS = 28–35 → Step 3へ

Step 3: 頭部感覚管
├── POC欠如 + POP欠如 → トウカイヨシノボリ (R. telma) [D1に縦長暗色斑列]
├── POC欠如 + POP通常あり → シマヒレヨシノボリ (R. tyoni) [尾鰭下半赤橙色♂]
└── POC・POP両方あり → Step 4へ

Step 4: 分布
├── 小笠原諸島 → オガサワラヨシノボリ (R. ogasawaraensis) [胸鰭基部に水平黒バー]
└── 本州〜琉球 → Step 5へ

Step 5: D1♂形状 + 体長
├── D1♂鎌型で伸長 + 頬に空色斑 + SL>60mm → ルリヨシノボリ (R. mizunoi)
├── D1♂烏帽子型で伸長 + SL>60mm → オオヨシノボリ (R. fluviatilis)
├── D1♂将棋駒型で低い + SL<40mm → トウヨシノボリ kurodai型
├── D1♂三角形/低い + 頬にミミズ状斑 → シマヨシノボリ (katonis/nagoyae)
└── D1♂低い + PDS極少 + 頬無斑 + 琵琶湖 → ビワヨシノボリ (R. biwaensis)（暫定）

Step 6: パイヌキバラヨシノボリ亜種間
├── 第5腹鰭軟条 3–4分枝(通常4) + D2翼状突起V10上 → R. a. aonumai（西表島）
└── 第5腹鰭軟条 2–3分枝(通常2) + D2翼状突起V9上 → R. a. ishigakiensis（石垣島）
```

## 形態分類の限界と注意点

1. **kurodaiタイプ標本の問題**: ホロタイプは損傷が激しく（33.9mm SL雄）、斑紋がほぼ不明瞭。Suzuki & Chen (2011)はsp. OR偽橙色型に一致するとしたが、遺伝的紐付けは不可能
2. **katonis = nagoyae?**: Suzuki & Chenはkatonis標本がsp. CB（シマヨシノボリ）に一致し、R. nagoyaeと同種の可能性を指摘。命名規約上の先取権問題が未解決
3. **fluviatilisの1個体問題**: ホロタイプ1個体のみのデータ。変異幅が不明
4. **mtDNAの信頼性**: 属全体でmtDNA浸透が確認されており、核DNAベースの同定が必須（Yamasaki et al. 2015）
5. **biwaensis未確認**: 原記載論文が入手できていないため、暫定データとして扱う
6. **sp.OR（トウヨシノボリ）の形態変異**: 橙色型・偽橙色型・宍道湖型・縞鰭型など多型的で、環境に応じて形態が大きく変動する。kurodaiの記載はこの一部の型にしか対応しない
