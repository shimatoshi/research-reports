## 目的

回遊型sp.ORの遺伝的独立性を検証するためのフィールドサンプリング。論文B（sp.OR内の回遊型/陸封型の遺伝的構造）の核心データを取得する。

## 基本方針

### 非致死的サンプリング

個体群の全容が不明な段階で大量に殺すべきではない。以下の方針で最小限の犠牲に留める。

| 処理 | 対象 | 目的 |
|------|------|------|
| **固定（致死）** | 各サンプリング地点で**1個体のみ** | 耳石Sr:Ca比による回遊履歴の物証 |
| **フィンクリップ（非致死）** | 残りの全個体 | DNA解析（sreb2ハプロタイプ確認） |
| **写真撮影** | 全個体 | 形態記録 |

### フィンクリップの手順

1. 採集した個体をバケツ等で一時保持
2. Google Pixelでスケール（定規）入りの記録写真を撮影
3. 尾鰭 or 胸鰭の端を眼科用ハサミで2-3mm切る（鰭は再生する）
4. 切片を個体ごとにラベル付き小瓶に入れ、99.5%エタノールで保存
5. 写真のファイル名と小瓶のラベルを対応させる（例: TEGA-01, TEGA-02...）
6. 魚をリリース

### 固定個体の処理

- 最も典型的な形態の**成熟オス**を選ぶ（体サイズ大、背鰭伸長、体色明瞭）
- 99.5%エタノールで全身固定
- こいつが形態の基準点になる
- 後日、耳石摘出 → 樹脂包埋 → 研磨 → Sr:Ca比測定（共同利用機器）

### 写真撮影の要件（現段階）

- Google Pixelで十分。深度合成は不要（将来的に博物館機材で再撮影する前提）
- **必ずスケール（定規）を入れる**
- 体側面（左右）、背面、腹面を撮影
- 第一背鰭の形状がわかるアングルを含める
- 可能なら婚姻色・斑紋パターンがわかるように

## サンプリング地点

### 地点1: 手賀沼水系（千葉県）— 練習 + 湖河回遊型データ

**目的**: 採集・フィンクリップ・撮影の手技の練習。同時に湖河回遊型sp.ORの所属問題を解く。

- 手賀沼からの流入河川（上大津川等）で遡上個体を狙う
- 先行観察で初夏〜夏に2-4cmの遡上個体を多数確認済み
- ただし成熟個体（6cm以上）も必要 → 河川中流域で採集

**期待されるデータ**:
- 手賀沼のsp.ORのsreb2ハプロタイプ
- 回遊型グループに入る → 湖河回遊 = 遺伝的に回遊型
- 陸封型グループに入る → 湖河回遊 = 陸封型が湖を利用しているだけ
- **どちらの結果でも論文に書ける**

**注意**: 手賀沼は海から隔離されているため、ここだけでは「真の両側回遊型」の証拠にならない。必ず海接続河川のデータが必要。

### 地点2: 海接続河川（千葉県南部 候補）— 本番

**目的**: 確実な両側回遊型sp.ORの採集。論文Bの核心サンプル。

- 太平洋側の海接続河川の本流の瀬
- 大型（6-8cm）のsp.ORを狙う
- 耳石確認で回遊履歴を物証として押さえる

**候補河川**: 未定。以下の条件で選定する:
- 海と直接接続（堰・ダムで遮断されていない）
- 本流に瀬がある（礫底）
- sp.ORの生息記録がある
- アクセス可能

### 地点間比較の論理

| 地点 | 環境 | sreb2が回遊型 | sreb2が陸封型 |
|------|------|-------------|-------------|
| 海接続河川 | 両側回遊が可能 | 予想通り（基準点） | 仮説の再考が必要 |
| 手賀沼 | 海隔離・湖河回遊 | 湖河回遊=回遊型 | 湖河回遊=陸封型 |

## 必要な装備

### 最低限（すぐ揃える）

- [ ] タモ網
- [ ] バケツ（一時保持用）
- [ ] 99.5%エタノール（薬局 or 試薬販売）
- [ ] 小瓶（スクリューキャップ付き、2ml程度）× 20本以上
- [ ] 眼科用ハサミ（先が細いもの。フィンクリップ用）
- [ ] ピンセット
- [ ] 油性ペン + マスキングテープ（ラベル用）
- [ ] 定規（写真スケール用）
- [ ] Google Pixel（持っている）

### あると良い（後から）

- [ ] MS-222（魚類用麻酔薬）— 写真撮影の質が上がる。博物館と繋がれば入手しやすい
- [ ] 実体顕微鏡 — 耳石摘出に必要。中古で1-3万円
- [ ] エポキシ樹脂 + 耐水ペーパー — 耳石の包埋・研磨用
- [ ] メス・解剖用具 — 耳石摘出用

## タイムライン

### Phase 1: 準備（4月中）

- 装備を揃える
- 手賀沼水系の下見（ポイント確認）
- 可能なら手賀沼で練習採集

### Phase 2: 手賀沼サンプリング（5-6月）

- 産卵期の成熟個体を狙う
- フィンクリップ + 写真
- 1個体固定

### Phase 3: 海接続河川サンプリング（5-7月）

- 候補地選定後、遡上～産卵期に合わせて採集
- フィンクリップ + 写真
- 1個体固定

### Phase 4: 解析・コンタクト

- 耳石摘出の練習（手賀沼の固定個体で）
- 深度合成写真 + 耳石断面 + フィンクリップを揃えた状態で博物館（生命の星等）にコンタクト
- DNA解析（sreb2等）の協力を依頼

## 証拠の構造（論文Bに向けて）

最終的に以下が揃えば論文として成立する:

1. **耳石Sr:Ca比**（各地点1個体）→ 回遊履歴の物証
2. **sreb2ハプロタイプ**（各地点N個体）→ 陸封型との遺伝的非共有の確認
3. **写真**（全個体）→ 形態記録、将来の種記載の素材
4. **既存GenBankデータとの比較** → Yamasaki et al. 2015の配列との照合

これらを持って博物館にコンタクトし、DNA解析の協力を得る。「実験だけお願いしたい、他は全部自分でやった」という形が理想。
