# Rhinogobius（ヨシノボリ属）

日本産ヨシノボリ属の研究レポート。

## 種フォルダ

| フォルダ | 和名 | メモ |
|----------|------|------|
| `all/` | 属横断 | 複数種にまたがる比較・総論 |
| `kurodai/` | クロダハゼ | 旧トウヨシノボリ偽橙色型。手賀沼水系メイン |
| `nagoyae/` | ナゴヤハゼ | 今後追加予定 |
| `unclassified/` | 未分類 | 旧トウヨシノボリ系など既存種に帰属できないもの |

## 各種フォルダの中身

- `papers/` — 学術論文ベースのレポート
- `field/` — フィールド観察・採集記録
- `web-verified/` — ネット情報で論文裏付け済み
- `unverified/` — 根拠未検証の情報
