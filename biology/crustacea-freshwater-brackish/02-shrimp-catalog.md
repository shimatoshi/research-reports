# 日本産淡水・汽水エビ類 種別カタログ

作成: 2026-04-17 / 対象: 本州〜琉球・北海道の淡水〜汽水域に生息するエビ類 + 在来/外来ザリガニ類
目的: 在野研究者のオフライン参照用種別ファクトシート。学名/和名/分布/生息環境/サイズ/両側回遊性/形態的特徴/保全状況を主要カラムとして整備。

---

## 0. 日本産淡水・汽水エビ相の全体像

日本の内水面に出現するエビ類(十脚目コエビ下目・イセエビ下目の一部)は、諸喜田(1979)以来の古典的整理を経て、現在は豊田ほか(2019)『日本産淡水性・汽水性エビ・カニ図鑑』(緑書房)や福家(日本の川えびチェックリスト)によって総計166種規模の甲殻類相としてまとめられている。主要分類群は以下のとおり。

- **ヌマエビ科 Atyidae**: ヌマエビ属 *Paratya*、カワリヌマエビ属 *Neocaridina*、ヒメヌマエビ属 *Caridina*、オニヌマエビ属 *Atyopsis* など。第1・2胸脚先端の"はさみ"がブラシ状で、付着藻類や有機物デトリタス摂取に特化。
- **テナガエビ科 Palaemonidae**: テナガエビ属 *Macrobrachium*、スジエビ属 *Palaemon*、シラタエビ属 *Exopalaemon*。第2胸脚が顕著に発達する捕食者/雑食者。
- **エビジャコ科 Crangonidae**: 汽水・海産中心で河口に侵入するグループ。
- **ザリガニ下目 Astacidea**: 在来 Cambaroides と外来 Procambarus・Pacifastacus。

生活史面では、**両側回遊(amphidromy)** — すなわち淡水で孵化したゾエア幼生が河川を流下して海(または汽水)で浮遊期を過ごし、稚エビで遡上定着する生活型 — が、南西諸島を含む黒潮系河川の甲殻類相を規定する最大の要因である。ヤマトヌマエビ・トゲナシヌマエビ・ヒメヌマエビ等のヌマエビ科、コンジンテナガエビ等のテナガエビ科は、この両側回遊を核にボウズハゼ属・ヨシノボリ属(Rhinogobius 汽水型)と同一の河川生態系モジュールを構成する。陸封型への転換は陸水湖沼や用水路定着個体群で何度か独立に生じている(例: スジエビ Aタイプ、淡水湖沼テナガエビ)。

## 1. ヌマエビ科 Atyidae

### 1.1 ヌマエビ *Paratya compressa* (De Haan, 1844)

| 項目 | 内容 |
|---|---|
| 和名 | ヌマエビ(小卵型) |
| 学名 | *Paratya compressa* (De Haan, 1844) |
| 分布 | 本州(新潟県・千葉県以南の西日本)、四国、九州、南西諸島、朝鮮半島南部 |
| 生息環境 | 河川下〜中流域、感潮域上限付近。流れのある砂礫底・植生帯 |
| サイズ | 雄 20 mm / 雌 25–30 mm |
| 両側回遊 | **あり**。ゾエアは海で発達、稚エビ期に遡上 |
| 主要形態 | 額角に上縁歯20前後、下縁歯2–4。体は半透明で細かな斑紋。第1・2胸脚に典型的な"筆"状房毛 |
| 保全状況 | 環境省レッドリスト非掲載だが、都府県RDBでは準絶滅危惧扱いの地域あり(京都府等) |

**分類史の要点**: Ikeda et al. (1999) 以前は *P. compressa compressa* の「北部・中部群(大卵型)」「南部群(小卵型)」と *P. c. improvisa*(ヌカエビ、大卵型)の2亜種3群に整理されていた。Ikeda (1999) は大卵型北部・中部群とヌカエビを統合して *Paratya improvisa* (ヌカエビ)、小卵型南部群を *P. compressa* (ヌマエビ狭義) とし、現在もこの2種体制が標準。小卵多産・両側回遊は *P. compressa* のみ、*P. improvisa* は大卵少産・純淡水完結型という生活史分化は、秋山(2016以降)の塩分応答実験でも裏付けられている。岩手県沿岸河川での近年の採集記録(Aquatic Animals, 2025)は北限更新に関わる重要な報告。

| 項目 | 内容 |
|---|---|
| 和名 | ヤマトヌマエビ |
| 学名 | *Caridina multidentata* Stimpson, 1860 |
| 分布 | 本州(房総半島・福井県以南)、四国、九州、南西諸島、台湾〜フィジー・マダガスカル |

**生活史**: 雌は抱卵2〜4週で孵化、ゾエアは海に下って9回の脱皮を経て約1か月で体長4 mmの稚エビに変態、夜間に底歩行で河口から遡上する(Hamano 1980s 系列の観察)。コケ取り能力からアクアリウム流通量が極めて多く、その供給圧は本州太平洋岸・九州の個体群に対して無視できない。

## 2. テナガエビ科 Palaemonidae(陸水性)

### 2.1 テナガエビ *Macrobrachium nipponense* (De Haan, 1849)

| 項目 | 内容 |
|---|---|
| 和名 | テナガエビ |
| 学名 | *Macrobrachium nipponense* (De Haan, 1849) |
| 分布 | 北海道南部〜九州、琉球、東アジア広域 |
| 生息環境 | 河川下流・汽水湖・淡水湖沼(3生態型分化) |
| サイズ | 雄 80–100 mm(はさみ脚込 >150 mm)/雌 60–80 mm |
| 両側回遊 | **原則あり**、ただし淡水湖沼型は陸封(大卵少産) |
| 主要形態 | 第2胸脚が顕著に伸長、成熟雄で体長を超える。頭胸甲の斑紋明瞭、額角は歯数多 |
| 保全状況 | 食用漁獲対象、広域普通種 |

**3型分化**: 本種には遺伝的に分化した(1)河川型(小卵多産・両側回遊、下流〜河口)、(2)淡水湖沼型(大卵少産・陸封、琵琶湖・霞ヶ浦等)、(3)汽水湖沼型(中卵中産、浜名湖等)の3エコタイプが知られ、卵サイズと塩分依存性の分化が発達している。繁殖期は5–9月、抱卵数 1,000–2,000、卵径 1 mm未満。

**諸喜田茂充の貢献**: 琉球大学・諸喜田茂充は1970年代以降、琉球列島陸水エビ類の分布・種分化を体系化し、コンジンテナガエビを含むテナガエビ科琉球産種群の基礎を築いた。西表島固有の**ショキタテナガエビ** *Macrobrachium shokitai*(大卵少産・陸封型、絶滅危惧I類)は諸喜田への献名(Fujino & Baba, 1973)。諸喜田ら(2003)は石垣島採集個体からカスリテナガエビを新標準和名として提唱、川原・諸喜田・成瀬(2019)はマングローブヌマエビの初期生活史を報告するなど、活動は現役で継続。佐伯ら(2019, CANCER 31)は琉球列島産ネッタイテナガエビ種群3種の分類・形態を再整理、成瀬貫らの Zootaxa / Crustaceana 系列の近年の論文も Macrobrachium 再整理の中核。

## 4. 両側回遊エビ–ボウズハゼ ニッチ群(生活史フォーカス)

琉球〜南日本の清流では、**両側回遊の一次消費者・デトリタス食者ギルド**が生態系の礎を成す:

- **付着藻類削り取り**: トゲナシヌマエビ、ヒメヌマエビ、ボウズハゼ *Sicyopterus japonicus*、ナンヨウボウズハゼ *Stiphodon* spp.
- **落葉デトリタス破砕**: ミゾレヌマエビ、ヤマトヌマエビ、サキシマヌマエビ
- **浮遊粒子ろ過**: オニヌマエビ(唯一の特殊化)
- **大型捕食**: コンジンテナガエビ、ヒラテテナガエビ

これらは全て、(1)産卵→孵化ゾエアが流下→海(汽水)で1か月前後プランクトン生活→稚エビ変態→河口から遡上、という共通生活史スケルトンを持つ。結果として、河川–海洋の縦断連続性(longitudinal connectivity)が失われる — 堰堤・ダム・河口閉鎖・埋立 — と、これら全ての種が同時に消える。沖縄本島北部・奄美・西表の保全単位としての価値はここに由来する。Hamano・諸喜田・成瀬・福家らの研究系列は、この生活史共通性を"魚類と甲殻類を一括するべき両側回遊保全単位"として提示している(成瀬 2010年代以降の Crustaceana / Zootaxa 記載シリーズ、水産学会誌)。

## 7. 今後の整理課題(在野研究者メモ)

1. **ヌマエビ属2種問題**: *P. compressa* × *P. improvisa* の雑種帯・接触帯の把握。秋山らの塩分応答実験を現地サンプルで追試したい
2. **カワリヌマエビ属の侵入**: *N. davidi* 系統の在来 *N. denticulata* への侵入・交雑実態は県レベルでほぼ未整理
3. **スジエビ B/C型**: 18S rDNA マーカーでの既存河川サンプル再検査の必要性
4. **琉球 Caridina**: *C. prashadi* / *C. rubella* / *C. brevirostris* の種境界は近年の DNA 解析で再整理中、最新 Zootaxa を継続フォロー
5. **両側回遊保全単位**: 魚類(ボウズハゼ・ヨシノボリ汽水型)との統合保全計画への接続 — 隣接する研究報告 `fish/` 以下との相互参照を想定
